Webhook

Written on 2025-20-12

Just a short note because I am feeling proud as I had an idea and actually realised it.

I thought that it is not of much convenience for me to use scp to copy files onto the server. git push is much more powerful. It provides with:

  • version control;
  • a free back-up;
  • it's cross-platform as is;
  • and arguably it's more convenient, at the very least, concise.

Idea is quite simple: as you push to github, it sends an http notification to server and it does smth. Pulls in my case. Took smth like 2 hours for me to make in spite DeepSeek's help.

Feel like I've learned a lot.
